Country register

The country register is a list of British English-language names and descriptive terms for countries.

View register

You can also download a copy of the register.

The country register contains 199 records and includes the following fields:

  • country (the country’s 2 letter ISO 3166-2 code)
  • name (the commonly used name of the country)
  • official-name (the country’s official name)
  • citizen-names (the name of the country’s citizens)
  • start-date (when the country was first recognised)
  • end-date (when the country was dissolved or changed its name)

Using this register

Download a copy of this register. You’ll need to regularly update it to make sure you have the most current version of the data.

If you’re building a service, you can use this data to create elements for forms.

You might need to combine data from this register with data from another source (such as the territory register) to do this.

If you do, you must make it clear the element has been built using more than just country data. For example, name the element ‘Choose location’ instead of ‘Choose country’.
